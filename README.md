## Descripción

Sitio web de dos páginas que simula una base de datos de películas y actores, inspirado en el diseño de TMDB (The Movie Database).

### Páginas incluidas

**`index.html` — Página de película**
Muestra la información de la serie *One Piece (2023)*: poster, géneros, sinopsis y detalles de producción. Incluye una sección de cast con tarjetas estilizadas como carteles de "Se Busca", con scroll horizontal.

**`luffy.html` — Página de actor**
Perfil del actor Iñaki Godoy, quien interpreta a Monkey D. Luffy. Incluye biografía, datos personales, filmografía detallada, proyectos destacados, premios y redes sociales.

---

## Conceptos aplicados

- **Flexbox** — para el layout del hero, la sección de cast y la info de la película
- **CSS Grid** — para el layout de dos columnas en la página del actor
- **Media Queries** — diseño responsivo 
- **Variables CSS (`:root`)** — paleta de colores centralizada y reutilizable
- **Google Fonts** — tipografías 
- **HTML semántico** — uso de `<article>`, `<section>`, `<header>`, `<aside>`, `<nav>`, `<footer>`
